Feedback Link

이 주소는 개별 피드백 링크 전용입니다.

선생님이 전달한 전체 URL로 접속해야 학생별 피드백 페이지가 열립니다. 루트 주소만 열거나 링크 일부가 빠지면 상세 페이지 대신 안내 화면이 표시됩니다.

정상 접속 방법

카카오톡이나 문자로 받은 링크를 처음부터 끝까지 그대로 열어 주세요. 학생명 슬러그가 포함된 주소여야 실제 피드백 페이지가 나옵니다.

STEP 1
전달받은 전체 링크를 확인 `/코딩쏙학원_학생_피드백/` 같은 경로가 URL 끝에 포함되어 있어야 합니다.
STEP 2
브라우저에서 그대로 열기 링크 일부만 복사되거나, 루트 주소만 열리면 상세 피드백 대신 안내 페이지가 열릴 수 있습니다.
STEP 3
잘못 열린 경우 새 링크 재요청 주소 오타나 만료 여부가 의심되면 발송한 쪽에 같은 학생의 최신 링크를 다시 요청해 주세요.
