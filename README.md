# 공유형 수업 피드백

`index.html`은 기존 PNG 다운로드형 템플릿을 URL 공유형 단일 페이지로 바꾼 버전입니다.

사용 흐름:

1. 이 파일을 공개 URL에 배포합니다.
2. 화면의 `공개 Base URL`에 그 주소를 입력합니다.
3. 학생 이름을 넣으면 `URL 이름`이 `코딩쏙학원_[이름]_피드백` 형식으로 자동 생성됩니다.
4. 피드백 내용을 작성하고 `공유 링크 만들기` 또는 `카톡으로 공유`를 누릅니다.

현재 구현 특징:

- 첫 페이지는 로고와 안내 문구가 보이는 표지이고, 다음 버튼을 누르면 실제 피드백 내용이 열립니다.
- URL 쿼리에 내용을 담아 링크만으로 같은 페이지를 다시 열 수 있습니다.
- 링크 경로를 `.../코딩쏙학원_홍길동_피드백?...` 형태로 생성합니다.
- 모바일에서는 Web Share API를 통해 카카오톡 공유 대상으로 보일 수 있습니다.
- `vercel.json`, `_redirects`를 같이 넣어 경로형 URL이 정적 호스팅에서도 열리게 준비했습니다.

제한:

- 실제로 카카오톡에서 열리게 하려면 공개 배포가 필요합니다.
- 내용이 너무 길면 URL 길이가 길어질 수 있습니다.
- `https://도메인/코딩쏙학원_홍길동_피드백` 형태로 보이더라도 실제 데이터는 쿼리 문자열에 담기므로, 내용이 매우 길면 링크가 길어집니다.
